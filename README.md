# Mass Tort Law Firm Website

A complete, production-ready mass tort law firm website built as a single self-contained `index.html` file.

## Features
- Hero section with CTA
- Practice Areas (pharmaceutical, medical device, environmental, product liability, data breach)
- Case Evaluation / Intake Form with validation
- About section with firm history & credentials
- Testimonials & Case Results
- FAQ section
- Contact section with consultation booking
- Blog / Resources
- Responsive design, SEO, accessibility (WCAG AA)

## Tech Stack
- Vanilla HTML5, CSS3, JavaScript (single file)
- Font Awesome icons via CDN
- No build step required

## Deploy
Push to Render for static site hosting.
